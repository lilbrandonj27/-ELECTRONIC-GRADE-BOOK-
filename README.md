# SmartTrack Gradebook System

## Overview

SmartTrack Gradebook System is a modern classroom management and student performance tracking application designed to help teachers efficiently manage grades, attendance, assignments, and student progress. The platform was created to solve common classroom challenges by providing a centralized, easy-to-use dashboard for educators.

This system is especially useful for teachers managing multiple classes and large groups of students. SmartTrack simplifies grading, tracks attendance in real time, and generates insightful reports to improve student success.

---

# Features

## Teacher Dashboard

* Clean and responsive interface
* Overview of all classes and students
* Quick access to assignments and grades
* Performance summaries and analytics

## Student Management

* Add, edit, and remove students
* Store student information securely
* Organize students by class period or subject
* Search and filter student records

## Gradebook Management

* Record assignment grades
* Calculate averages automatically
* Weighted grading support
* GPA and percentage calculations
* Grade editing and updating

## Attendance Tracking

* Daily attendance management
* Present, absent, and tardy tracking
* Attendance history logs
* Attendance summaries for each student

## Assignment Tracking

* Create assignments and due dates
* Track completion status
* Manage assignment categories
* View missing assignments

## Reports and Analytics

* Student performance reports
* Class average statistics
* Attendance analytics
* Progress tracking charts

## Secure Authentication

* Teacher login system
* Password protection
* Secure user sessions

# Installation Guide

## 1. Clone the Repository

```bash
git clone https://github.com/yourusername/smarttrack-gradebook.git
cd smarttrack-gradebook
```

## 2. Create a Virtual Environment

### Windows

```bash
python -m venv venv
venv\Scripts\activate
```

### Mac/Linux

```bash
python3 -m venv venv
source venv/bin/activate
```

---

## 3. Install Dependencies

```bash
pip install -r requirements.txt
```

---

## 4. Run the Application

```bash
python app.py
```

The application will start locally at:

```bash
http://127.0.0.1:5000/
```
